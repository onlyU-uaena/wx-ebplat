# Taro 美家微信小程序

该项目使用taro小程序混合开发，建议使用taro的react语法开发

### 技术栈:

* Typescript
* React
* Redux
* Taro
* 原生小程序语法

[Taro文档](http://taro-docs.jd.com/taro/docs/README)

[Taro代码与小程序混写](http://taro-docs.jd.com/taro/docs/hybrid)

### UI

* [TaroUI](https://taro-ui.jd.com/#/)

### 代码规范

* eslint

### 包管理系统

* yarn

### 开发前准备
确保node环境在8.0以上
需要全局安装taro cli
需要安装yarn

在项目根目录运行下列代码来开始开发
```
yarn
yarn run dev:weapp
```
